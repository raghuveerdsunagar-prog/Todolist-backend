# Todolist-backend
Todolist taskmanagement where user can add manage task. its todolist backend includes nodejs prsima where you can see all api endpoints and logics of todlist



This is the backend API for the Todolist application.  
It provides RESTful endpoints to manage tasks (create, read, update, delete).  
The backend is built with Node.js, Express, and connects to a database (e.g.,MySQL).

---

## Features
- Add new tasks
- Get all tasks
- Update tasks (mark complete, edit text)
- Delete tasks
- RESTful API design
